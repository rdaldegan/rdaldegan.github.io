| Conecta Brasal

2024 — Atual

Contexto

Problema: A Brasal, distribuidora Coca-Cola no DF e região, precisava centralizar todo o relacionamento com seus clientes B2B em uma única plataforma — desde pedidos de produtos até pagamento de dívidas e solicitação de equipamentos.

Usuários: Clientes B2B (bares, restaurantes, mercados), equipe comercial e administração interna da Brasal.

Minha atuação

  • Backend: APIs REST, regras de negócio, integração com sistemas internos
  • Frontend: interfaces do app mobile e painel web
  • Infraestrutura: configuração de containers e orquestração com Docker Swarm

Tecnologias

Laravel 10 React Native React.js MySQL Docker Swarm

Arquitetura

Sistema dividido em 3 microsserviços independentes, cada um com seu backend Laravel:

  • Usuários: autenticação, cadastro e gestão de perfis de clientes
  • Vendas: catálogo de produtos, carrinho, pedidos e pagamento de dívidas
  • Campanhas: painel administrativo, promoções e gestão de campanhas comerciais

Deploy com Docker Swarm para orquestração de containers, com réplicas e load balancing para garantir disponibilidade e escalabilidade.

Desafios técnicos

  • Garantir alta disponibilidade para um sistema que processa todas as vendas Coca-Cola da região
  • Orquestrar 3 módulos independentes com comunicação consistente entre eles
  • Escalar horizontalmente com Docker Swarm e load balancing entre réplicas de containers

Solução

Arquitetura em microsserviços com separação clara de responsabilidades entre os módulos de Usuários, Vendas e Campanhas. Cada serviço roda em containers Docker gerenciados pelo Swarm, que cuida da replicação automática e distribuição de carga. Isso permitiu escalar cada módulo de forma independente conforme a demanda.

Resultados

  • 12.000+ clientes cadastrados na plataforma
  • R$ 50M+ de faturamento médio por semana
  • Plataforma única para todo o fluxo comercial B2B da Brasal no DF
  • Disponível em app mobile (iOS/Android) e versão web

| Espaço Cliente — Brasal Incorporações

2025 — Atual

Contexto

Problema: Compradores de imóveis da Brasal Incorporações precisavam de um canal unificado para acompanhar seus empreendimentos, pagar parcelas, agendar visitas e abrir chamados de assistência técnica — tudo de forma digital.

Usuários: Clientes compradores de imóveis da Brasal Incorporações.

Minha atuação

  • App mobile completo em React Native (64 telas)
  • Dashboard web em Vue.js 3
  • API REST em Laravel Lumen com integração a sistemas legados SOAP
  • Integração com banco Oracle legado e MySQL

Tecnologias

React Native Expo Vue.js 3 Laravel Lumen MySQL Oracle DB AWS S3 Salesforce OneSignal Docker

Arquitetura

Sistema distribuído em 3 camadas independentes: app mobile (React Native/Expo), dashboard web (Vue.js 3) e API backend (Laravel Lumen). A API funciona como ponte entre os frontends modernos e os sistemas legados da empresa via SOAP (32+ operações WSDL).

  • Dual Database: MySQL para dados do app + Oracle para dados legados de contratos e financeiro
  • Integrações: Salesforce (CRM), OneSignal (push), AWS S3 (documentos), SendGrid (e-mails)
  • Autenticação: JWT com suporte a biometria (Face ID / impressão digital) no mobile

Desafios técnicos

  • Integrar com web services SOAP legados (20+ anos) sem comprometer a experiência do usuário
  • Manter dois bancos de dados (MySQL + Oracle) sincronizados e consistentes
  • Cálculos financeiros complexos: juros por atraso, antecipação de parcelas, simulações de desconto
  • Autenticação biométrica cross-platform (Android e iOS)
  • Notificações push em escala para milhares de compradores

Solução

Construí uma camada de abstração SOAP no backend que traduz chamadas REST em operações WSDL, com cache para reduzir latência. O padrão Repository + Service isola o acesso aos dois bancos de dados, permitindo migrar gradualmente do Oracle para o MySQL. No mobile, um design system próprio com 40+ componentes garante consistência visual, e o Recoil gerencia o estado de forma atômica. Filas de jobs processam notificações e e-mails de forma assíncrona.

Resultados

  • App disponível em iOS e Android + dashboard web
  • Fluxo completo: da consulta do empreendimento ao pagamento de parcelas
  • Integração com Salesforce para agendamentos e assistência técnica
